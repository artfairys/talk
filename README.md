## README

[![Greenkeeper badge](https://badges.greenkeeper.io/lizheming/anychat.svg)](https://greenkeeper.io/)

application created by [ThinkJS](http://www.thinkjs.org)

## install dependencies

```
npm install
```

## start server

```
npm start
```

## deploy with pm2

```
pm2 startOrReload pm2.json
```
